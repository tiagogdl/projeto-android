# Projeto Android

Site sobre a história e evolução do sistema operacional Android, desenvolvido com foco em aprendizado prático de HTML e CSS.

🔗 [Ver deploy](https://tiagogdl.github.io/projeto-android/)

---

## 📋 Sobre

Projeto desenvolvido para praticar HTML e CSS na construção de interfaces modernas e responsivas. O site apresenta a evolução do Android de forma visual e organizada, com integração de vídeos do YouTube e adaptação para diferentes dispositivos.

## ✨ Funcionalidades

- Linha do tempo visual das versões do Android
- Integração de vídeos do YouTube
- Layout totalmente responsivo
- Links externos e imagens otimizadas

## 🚀 Tecnologias

- HTML5
- CSS3
